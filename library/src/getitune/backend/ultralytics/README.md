## Supported Tasks

| Task | Description |
|------|-------------|
| Detection | Object detection with bounding boxes |
| Instance Segmentation | Instance segmentation with masks and bounding boxes |

**2 supported tasks.**

## Supported Models

| Model | Task | Variants |
|-------|------|----------|
| YOLO26-N | Detection | `yolo26n` |
| YOLO26-S | Detection | `yolo26s` |
| YOLO26-M | Detection | `yolo26m` |
| YOLO26-N Seg | Instance Segmentation | `yolo26n-seg` |
| YOLO26-S Seg | Instance Segmentation | `yolo26s-seg` |
| YOLO26-M Seg | Instance Segmentation | `yolo26m-seg` |

**6 supported models.**
